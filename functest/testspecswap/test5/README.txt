This test runs the SpecSwap-RMC progrma using the specified input file.
The resulting outputs are compared to stored references.

Run the test by executing the "runtest.bash" script.

To look at the results of the fit, after creating a directory
named "./ANALYSIS", run test SpecSwap-RMC program again with the
"analysis.inp" as input file, and plot the curves with the provided
gnuplot input file "gnu.plot".

Please see the README.txt file in the SpecSwap-RMC root directory for
further details.
